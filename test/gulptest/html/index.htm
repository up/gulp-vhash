All things change.

Seit 1999 unterstütze ich Unternehmen beim Aufbau von flexiblen und wartbaren Frontend-Architekturen, dem Refactoring von Legacy Code sowie der Entwicklung von komplexen Lösungen.

Mit einer umfangreichen Sammlung an - meist selbstentwickelten - Analysetools finde ich schnell Schwachstellen und Optimierungpotential hinsichtlich Performance und Clean Code.

Weitere Schwerpunkte bilden die Entwicklung von individuellen Deployment-Löungen sowie Multi-Device Anwendungen mit CSS Media-Queries und dynamische geladenen Assets.
